Velvet Essence | Perfumería de Autor

Entregable: APF1 — Desarrollo Web con Bootstrap
Versión: 1.0
📝 Descripción del Proyecto

Velvet Essence es una landing page diseñada para una casa de perfumería exclusiva. El proyecto se centra en la elegancia y el minimalismo, utilizando Bootstrap 5.3 para garantizar una experiencia responsiva y sofisticada. Se ha priorizado la jerarquía visual y la legibilidad para reflejar la identidad de una marca de lujo.
🛠️ Tecnologías y Recursos

Para la implementación de este sitio se han utilizado los siguientes recursos externos vía CDN:

    Framework CSS: Bootstrap v5.3.0

    Bundle JS: Bootstrap JS v5.3.0

    Iconografía: Bootstrap Icons v1.13.1

    Tipografía: Google Fonts (Montserrat)

✅ Checklist de Componentes (Rúbrica ATF1)

A continuación se detallan los componentes de Bootstrap implementados según los requerimientos obligatorios:

    [x] Navbar Responsive: Barra de navegación fija (fixed-top) con menú colapsable y botón toggler para dispositivos móviles.

    [x] Layout con Grid System: Implementación de contenedores, filas y columnas responsivas (container, row, col-*) en múltiples secciones.

    [x] Cards de Producto: 8 tarjetas estilizadas (4 hombres / 4 mujeres) con imágenes, descripciones y botones de acción.

    [x] Tabla Estilizada: Tabla minimalista de "Intensidad y Fijación" con clases table, table-hover y table-borderless.

    [x] Imágenes Responsive: Uso de la clase img-fluid en banners informativos para adaptación automática.

    [x] Formulario de Contacto: Formulario con 4 campos obligatorios (Nombre, Email, Asunto, Mensaje) y validación nativa.

    [x] Modales Funcionales: - Modal de confirmación de envío.

    [x] Bootstrap Utilities: Uso extensivo de utilidades de espaciado (p-*, m-*), tipografía (text-*), visibilidad (d-none, d-sm-block) y decorativas (shadow, rounded-0).
